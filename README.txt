INSTRUCTIONS TO RUN:
===================
This project was created in IntelliJ but converted to be compatible also with Eclipse.
It has been tested to be working in both IDEs. No IDE is required to run the tests.
If maven is installed, you should be able just able to run the test by running the command below
from the location of the pom.xml

command: mvn clean compile test

All jars needed for this project are added to POM.xml
Jars  will be automatically downloaded during runtime.
For this to work maven must be locally installed and configured.
For help, please refer to this link:
http://maven.apache.org/install.html

------------------------------------
OR Select the TEST Class file(s)
- Right click
- Run as TestNG Tests

NOTE: chromedriver is included in the resource folder and path is configured accordingly.
